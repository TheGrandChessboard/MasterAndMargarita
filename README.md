# The Master and Margarita

A Python application inspired by Mikhail Bulgakov's masterpiece novel "The Master and Margarita".

## Description

This is a modern Python application that serves as a tribute to one of the greatest novels of the 20th century. The application currently displays the book's title and can be extended for various literary-related functionalities.

## Requirements

- Python 3.8 or higher

## Installation

Clone the repository:

```bash
git clone https://github.com/TheGrandChessboard/MasterAndMargarita.git
cd MasterAndMargarita
```

## Usage

Run the application directly:

```bash
python3 main.py
```

Or install and run as a package:

```bash
pip install -e .
master-and-margarita
```

## Development

### Running Tests

```bash
python3 -m pytest
```

### Code Quality

This project uses modern Python development tools:

- Type hints for better code clarity
- Docstrings for documentation
- Modern string formatting (f-strings)
- Entry point pattern with `if __name__ == "__main__"`

## About "The Master and Margarita"

"The Master and Margarita" is a novel by Russian writer Mikhail Bulgakov, written in the Soviet Union between 1928 and 1940. It is considered one of the best novels of the 20th century, as well as the foremost of Soviet satires.
